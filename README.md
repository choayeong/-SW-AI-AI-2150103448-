# 무역 용어 마스터 퀴즈 앱 (Trade Term Master Quiz App)

## 프로젝트 개요
무역·통상 전공자나 관련 분야에 관심 있는 학습자들이  
핵심 무역 용어를 쉽고 재미있게 익힐 수 있도록 제작한 웹 기반 학습 퀴즈 앱입니다.  
AI 없이도 브라우저에서 실행되며, 사용자 입력 데이터를 localStorage에 저장해 누적 점수와 오답을 관리할 수 있습니다.

---

## 주요 기능
- **무작위 퀴즈 출제**  
  무역 용어 정의(설명)를 화면에 표시하고, 사용자가 해당 용어를 입력하는 형식  
- **정답 확인 피드백**  
  - 정답 시 초록색 `"정답입니다! (+1점)"` 표시  
  - 오답 시 빨간 `"오답입니다. 정답: [용어]"` 메시지 출력  
- **점수 누적 저장**  
  localStorage를 사용해 개인 누적 점수 유지  
- **학습 완료/재시작 기능**  
  10문제 완료 후 “총점” 출력 및 다시 시작 버튼  

---

## 기술 스택
- HTML5 (퀴즈 UI)
- CSS (기본 레이아웃 및 폰트 적용)
- JavaScript (퀴즈 로직 및 점수 저장)
- localStorage (데이터 저장)
- Google Chrome (테스트 환경)
